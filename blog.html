Sesión 5

12-Septiembre-2023 3 Semana 7mo Ciclo

Quinta sesión, en este la sesión trata de la pagina Web donde se aprende añadir o a quitar cosas, cambiar de color las palabras. Además de añadir los links de contacto. Se deja un trabajo donde es mostrar el avance de la Bitácora Virtual.

14-Septiembre-2023 3 Semana 7mo Ciclo

Sesión 8

21-Septiembre-2023 4 Semana 7mo Ciclo

Octava sesión, se utilizo un programa 3D en manera de practica donde se hizo un modelado personalizado al inicio y después se midió el celular para así pasar el celular al programa ahora con medidas, se deja un trabajo donde se debe hacer mas piezas del celular.

Sesión 9

26-Septiembre-2023 5 Semana 7mo Ciclo

La novena sesión es donde se utilizo la pagina Onshape para esta vez usar una nueva herramienta donde se pueden realizar múltiples formas. Se dejo un proyecto de hacer una botella de cualquier tipo con color usando dicha herramienta enseñada.

Sesión 10

28-Septiembre-2023 5 Semana 7mo Ciclo

La décima sesión se muestra al profesor el celular hecho en Onshape con los accesorios lo cual se hicieron como seis accesorios, Popsocket, Cargador con USB, Audífonos, Case y Porta celular. Además de enseñar la lista de materiales para el proyecto Verbex.

Sesión 11

03-Octubre-2023 6 Semana 7mo Ciclo

La onceava sesión, es donde se usa la pagina de Tinkercad donde se explica el uso de este de lo que se aprendió en Laboratorio de Investigación 1. Luego de ello se enseño sobre en como se maneja los códigos para que funcione la LED como el usuario quiere hasta que se deja un trabajo donde todas las LED tienen que prenderse cada una para finalmente prenderse todas para la siguiente clase.

Sesión 12

5-Octubre-2023 6 Semana 7mo Ciclo

La doceava sesión se presenta el trabajo grupal que se dejo el día martes. Además de un nuevo proyecto que se trata sobre prender el Visualizador de 7 Segmentos, donde esta tenía que marcar desde el 0 al 3. Primero este se haría en el Tinkercad para saber como funciona primero para luego ser pasado al Arduino donde se buscaría un código para que el Visualizador marque del 0 al 3.

Sesión 13

10-Octubre-2023 7 Semana 7mo Ciclo

En esta sesión se aprendió en como se hace para conectar un motor, tanto las velocidades del motor tanto positiva como negativa. Luego de ello se dejo un trabajo donde el motor tendría que ir tanto en positivo como negativo y añadir otro motor para la siguiente clase.

Sesión 14

12-Octubre-2023 7 Semana 7mo Ciclo

La sesión trata de hacer el código para dichos motores que nos dio el profesor en el Tinkercad, en estos se haría funcionar un robot siendo lo primero en que mueva los ojos haciendo que parpadee y que luego de un guiño. Lo segundo es que se haga es que el robot avance para que luego retroceda para finalmente es que avance y vaya al lado izquierdo.

Sesión 15

17-Octubre-2023 8 Semana 7mo Ciclo

En esta sesión llevamos con el uso del Ultrasonido donde el profesor nos muestra en cómo usarlo, en cómo funciona y sobre todo en cómo se lee el texto cuando lejos o esta cerca de un rango especificado. También se pudo prender una LED utilizando el sensor, se dejo un trabajo donde se tenían que prender una LED cuando este lejos, dos LED cuando estén en medio y tres LED cuando estén cerca.

Sesión 16

19-Octubre-2023 8 Semana 7mo Ciclo

En esta sesión el profesor entrego en Tinkercad el circuito donde tendríamos que hacer el código, pero ahora utilizando el sensor, el primero donde tenia que ir dirigido hacia la pared y con el sensor detectaría antes de chocar para que este diera la vuelta, luego de ello iba dirigido hacia la puerta. La segunda trata de que tenía que detectar ambas cajas y cruzar por el único camino.

Sesión 17

24-Octubre-2023 9 Semana 7mo Ciclo

Sesión 18

26-Octubre-2023 9 Semana 7mo Ciclo
